# Calculate

## Опис проєкту

Calculate — це простий вебпроєкт калькулятора, створений за допомогою HTML, CSS та JavaScript.

Проєкт містить сторінку калькулятора, стилі оформлення та JavaScript-файл для реалізації логіки роботи.

## Структура проєкту

* `index.html` — основна HTML-сторінка проєкту.
* `style.css` — файл зі стилями для оформлення інтерфейсу.
* `script.js` — файл із JavaScript-кодом для роботи калькулятора.

## Як запустити проєкт

1. Склонуйте репозиторій на комп’ютер.
2. Відкрийте папку проєкту.
3. Знайдіть файл `index.html`.
4. Відкрийте файл `index.html` у браузері.

Додаткові залежності для запуску не потрібні.

## Призначення проєкту

Проєкт можна використовувати як навчальний приклад для вивчення базової взаємодії HTML, CSS та JavaScript у вебзастосунку.

## Автор

Проєкт використано для виконання лабораторної роботи з теми Pull Request та баг-репорт.
